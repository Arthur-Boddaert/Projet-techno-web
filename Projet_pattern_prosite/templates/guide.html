Guide d'utilisation de la version web

Une fois sur la page de l’application, compléter les deux chemins d’accès vers vos fichiers.
Le premier à compléter est le chemin vers le fichier avec les séquences dont vous voulez trouver le pattern.
Le second correspond au chemin d’accès vers le fichier contenant l’unique séquence dont vous voulez vérifier la présence du pattern à l’intérieur.
Enfin la partie seuil accepte un nombre entre 1 et la longueur des séquences et il permet de définir le nombre maximum d’acides aminés différents que le pattern peut afficher à chaque position avant de les remplacer par un X.
Pour lancer le programme cliquer sur le bouton “soumettre”.

Guide d'utilisation de la version en lignes de commandes

Les scripts peuvent également être retrouvés ici.

Paramètres :
Obligatoire et dans cet ordre :
1) Nom / chemin vers le fichier multifasta
2) Nom / chemin vers le fichier contenant la séquence
Optionnel :
-t or --threshold_value ; int ; Modifie le nombre d'acides aminés différents acceptés à une position avant de mettre un "x". Défaut : 4
Fonctionnement du programme :
Le programme pratt commence par extraire les séquences dans un dictionnaire avec Multifasta.sequences() puis le programme utilise MultipleAlignment pour créer un objet MultipleAlignment à partir du Enfin, il crée un objet de la classe PrositePattern et utilise la méthode search() pour obtenir le tuple.
Le programme utilise le tuple et le motif prosite obtenu avec la classe PrositePattern pour renvoyer la chaîne de caractères :'Pattern {prosite_pattern} found at position {position} : {match}' si une occurrence est trouvée sinon il renvoie 'Pattern {prositepattern} not found'.
Exemples de lignes de commandes :
Linux :
$ python3 pratt.py exemple1.fasta seq.fasta
$ python3 pratt.py exemple1.fasta seq.fasta -t 5
$ python3 pratt.py exemple1.fasta seq1.fasta -t 2
Windows :
> python.exe pratt.py exemple1.fasta seq.fasta
> python.exe pratt.py exemple1.fasta seq.fasta -t 5
> python.exe pratt.py exemple1.fasta seq.fasta -t 2
